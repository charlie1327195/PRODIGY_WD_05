# PRODIGY_WD_05 - Weather App

## 📌 Task Description
Build a weather web application that fetches weather data from a weather API based on the user’s location or a user-inputted location and displays the current weather conditions, temperature, and other relevant information.

---

## 🚀 Features
- Real-Time Weather Information
- Search Weather by City Name
- API Integration
- Temperature Display
- Weather Condition Display
- Responsive Design
- Interactive User Interface

---

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- Weather API
